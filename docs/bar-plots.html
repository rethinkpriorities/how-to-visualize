3 Bar plots

3.1 Simple bar plot

This is a simple bar plot2.

p <- ggplot(mpg, aes(x = class, fill = class)) +
  geom_bar(alpha = .75) +
  scale_fill_viridis(discrete = TRUE, option = "mako", end = .95) +
  theme_rethink()

ggplotly(p, tooltip = "x")
